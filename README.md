# palanaeum-learning

## Build Instructions

1. Clone repository to your personal machine
2. Add .env file to repository clone
3. Download and install Docker Desktop
4. In the terminal, navigate to repository
   use “docker compose up” to build application
5. Point browser toward localhost:3333 and the AdonisJS application will be there.

### Database Instructions

- the information for connection is in the .env file
- the application will create the database as part of the build
- the database is hosted at localhost:3306
